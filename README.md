Instalação da Extensão


Após fazer o clone do repositório, é necessário coloca-la dentro do google exxtensão e para isso, é necessário fazer alguns passos para ter exito.
-Instalar o vite (Abrindo o terminal na pasta principal e adicionanado esses códigos npm instal; npm run dev)
-Fazer o build da extensão (Abrindo tambem o terminal na pasta principal e escrever esse código npm build)

Após isso, já podemos adiciona-la e para isso, siga os passos.
-Abrir o google extensão
-Colocar para modo desenvolvedor
-Carregar sem compactação
-Procurar o arquivo e adicionar a pasta Dist

Depois desses passos, a extensão já se encontrará no Google Extensão.


Rodar o server Proxy:
Já que a extensão já se encontra no Google Extensão, agora é a hora de rodar o server.
Na pasta que se encontra o backend_jet_sales você irá abrir o terminal e escrever o seguinte código
-node index.js


Após esses passos, a sua extensão estará 100% funcional!
